# AI Cold Email Generator 🚀

An AI-powered web application that generates customized, professional cold emails by matching a user's skills and projects to a specific job or research role. Built for the GDG on Campus IIT Bhilai Build Sprint.

## 🛠️ Tech Stack
* **Frontend:** Streamlit
* **AI Engine:** Google Gemini API (`gemini-pro`)
* **Database:** Firebase Cloud Firestore
* **Hosting:** Hugging Face Spaces (Docker)

## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shubhamasare70-spec/email-generator
   2. instal dependencies
   pip install -r requirements.txt
   3'Database Setup:
    Place your firebase_key.json file in the root directory.
   4. run the app
   python -m streamlit run app.py
